# Code of Conduct

## Our pledge

We are committed to making participation in this project a harassment-free
experience for everyone.

## Our standards

Examples of behavior that contributes to a positive environment include:

- Showing empathy and kindness
- Respecting differing opinions and experiences
- Giving and accepting constructive feedback
- Taking responsibility and learning from mistakes

Examples of unacceptable behavior include:

- Harassment or discriminatory language
- Trolling, insulting, or derogatory comments
- Public or private harassment
- Publishing others' private information without consent

## Enforcement responsibilities

Project maintainers are responsible for clarifying and enforcing these standards.

## Scope

This code of conduct applies in all project spaces and public spaces where a
person is representing the project.

## Reporting

Report incidents by opening a private security report or contacting maintainers.

## Enforcement

Maintainers may remove, edit, or reject comments, commits, code, issues, and
other contributions not aligned with this Code of Conduct.
